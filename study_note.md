# Expo Study Notes

## Introduction

Expo is an open-source platform for making universal native apps for Android, iOS, and the web with JavaScript and React.

## Key Features

*   **Cross-platform:** Write one codebase that runs on multiple platforms.
*   **Fast development:** Use Expo Go to see your changes instantly.
*   **Easy to use:** Get started quickly with a simple setup process.
